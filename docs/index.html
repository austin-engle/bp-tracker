BP Tracker

Your simple, modern solution for tracking blood pressure readings anywhere.

About the App

BP Tracker helps you monitor your systolic, diastolic, and pulse readings over time. It features both a web interface and a dedicated iOS application.

The backend is powered by a serverless Go application running on AWS Lambda with data stored securely in RDS.

View on GitHub

Key Features

  • Effortless submission of Systolic, Diastolic, and Pulse readings.
  • Visualize your history and track trends with intuitive charts.
  • Automatic calculation of 7-day, 30-day, and overall averages.
  • Instant blood pressure classification based on the latest guidelines.
  • Seamless access via Web and dedicated iOS app.
  • Secure cloud storage powered by AWS.

Technology Stack

  • Backend: Go (Gin), AWS Lambda, API Gateway, RDS PostgreSQL, Secrets Manager
  • Frontend (iOS): Swift, SwiftUI, Swift Charts
  • Infrastructure: Terraform, Docker
  • This Site: HTML, CSS (via GitHub Pages)

BP Tracker

Take control of your health. The simple, secure, and modern way to monitor your blood pressure anytime, anywhere.

Take Control of Your Health Data

Understanding your blood pressure is key to managing your health. BP Tracker provides an easy-to-use interface on both iOS and the web to log your systolic, diastolic, and pulse readings.

Your data is stored securely using modern cloud technology (AWS Lambda, RDS), ensuring privacy and accessibility whenever you need it.

Explore the Code on GitHub

Why BP Tracker?

This project started as a practical tool for personal health monitoring and evolved into a demonstration of building a full-stack application using modern technologies like Go, SwiftUI, AWS serverless, and Terraform.

It aims to be a simple, reliable, and open-source solution for anyone needing to track their blood pressure readings effectively.

Unlock Key Insights

  • Effortless Tracking: Quickly submit Systolic, Diastolic, and Pulse readings from any device.
  • Visualize Trends: See your history at a glance with intuitive charts (iOS).
  • Automatic Averages: Instantly know your 7-day, 30-day, and overall averages.
  • Understand Your Numbers: Get immediate blood pressure classification based on the latest guidelines.
  • Cross-Platform Access: Use the dedicated iOS app or the simple web interface.
  • Secure & Private: Your health data is safe with robust AWS cloud storage.

Technology Stack

  • Backend: Go (Gin), AWS Lambda, API Gateway, RDS PostgreSQL, Secrets Manager
  • Frontend (iOS): Swift, SwiftUI, Swift Charts
  • Infrastructure: Terraform, Docker
  • This Site: HTML, CSS (via GitHub Pages)
